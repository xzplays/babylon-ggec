# Babylon.js 离线园区场景服务

这是一个使用 Microsoft Babylon.js 构建的园区漫游场景。项目可以在本地开发，也可以编译成 `dist/` 静态文件后离线打开或交给任意静态文件服务器托管。

## 功能

- 现实比例园区地图：场景约定 `1 Babylon 单位 = 1 米`，道路、建筑、广场和景观湖都按真实校园尺度重新放大。
- 可进入建筑：每栋建筑由墙体、屋顶、地面和门洞组成，不再是实心盒子；内部包含大厅、接待台、桌椅和隔断等真实可漫游空间。
- 快捷建筑列表：右侧面板可一键传送到建筑门前，或直接进入建筑内部大厅。
- 近场交互：靠近绿色入口标记后按 `E` 可以进入对应建筑。
- 漫游体验：支持第一视角、第三视角和俯瞰视角。
- 操控方式：`WASD` / 方向键始终按当前视图方向移动，鼠标拖拽环视，`Shift` 加速，`V` 切换视角。
- 离线构建：依赖安装后运行 `npm run build`，产物在 `dist/`。

## 使用

```bash
npm install
npm run dev
```

开发服务启动后在浏览器打开终端显示的本地地址。

## 编译离线产物

```bash
npm run build
```

编译完成后，`dist/` 目录就是可分发的静态园区场景服务。
